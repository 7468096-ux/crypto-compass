# Portfolio Builder - Документация

## Обзор
Portfolio Builder - это компонент для создания и расчета криптовалютных портфелей с разными уровнями риска.

## Функциональность

### 1. Три типа портфелей

#### Conservative (Консервативный)
- **Риск:** Низкий
- **Распределение:**
  - 60% Bitcoin (BTC)
  - 30% Ethereum (ETH)
  - 10% Stablecoins (USDT)
- **Подходит для:** Долгосрочных инвесторов, минимизация рисков

#### Balanced (Сбалансированный)
- **Риск:** Средний
- **Распределение:**
  - 40% Bitcoin (BTC)
  - 30% Ethereum (ETH)
  - 20% Solana (SOL)
  - 10% Other (разные альткоины)
- **Подходит для:** Баланс между стабильностью и ростом

#### Aggressive (Агрессивный)
- **Риск:** Высокий
- **Распределение:**
  - 30% Bitcoin (BTC)
  - 25% Ethereum (ETH)
  - 25% Solana (SOL)
  - 20% Altcoins (BNB и другие)
- **Подходит для:** Максимальный потенциал роста

### 2. Калькулятор инвестиций
- Ввод любой суммы инвестиции
- Быстрый выбор: $500, $1,000, $5,000, $10,000
- Автоматический расчет распределения

### 3. Расчет портфеля
Для каждой криптовалюты показывается:
- **Сумма инвестиции** в USD
- **Текущая цена** из CoinGecko API
- **Количество монет** которое можно купить
- **Процент** от общего портфеля

### 4. Визуализация
- Цветная шкала распределения активов
- Индикаторы уровня риска
- Легенда с процентами

## Технические детали

### Файлы
- `components/PortfolioBuilder.tsx` - основной компонент
- `app/page.tsx` - обновлен для интеграции компонента

### Интеграция с API
Компонент использует существующий API endpoint `/api/prices` для получения актуальных цен криптовалют:
- Автоматическое обновление каждую минуту
- Поддержка до 50 топ криптовалют
- Кэширование для оптимизации

### Технологии
- **TypeScript** - типизация
- **React Hooks** - useState, useEffect
- **Tailwind CSS** - стилизация (dark theme)
- **Lucide Icons** - иконки

## Использование

```tsx
import PortfolioBuilder from '@/components/PortfolioBuilder';

<PortfolioBuilder />
```

Компонент полностью самодостаточен и не требует дополнительных пропсов.

## Disclaimer
Компонент включает предупреждение о том, что это не финансовый совет и инвестиции в криптовалюты несут риски.
